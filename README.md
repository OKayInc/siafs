# siafs
SIA FUSE Filesystem, sponsored by the SIA Foundation.

## Status
Not working yet.

## Installing
siafs uses `CMake`.

### From source

    git clone https://github.com/OKayInc/siafs
    cd siafs
    mkdir build
    cd buid
    cmake ..
    make
    make install

#### Requirements
* libcjson
* libcurl
* libfuse 2.9+
